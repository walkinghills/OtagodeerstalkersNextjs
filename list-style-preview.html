Research basis: DOC NZ · Patagonia · NZDA National · FT Weekend — all use no decorative markers. Structure comes from spacing or rules, not icons.
Option A

Pure Spacing

No marks, no rules. Items breathe through gap alone. Cleanest possible — exactly what DOC NZ and Patagonia use.

Option B

Hairline Row Rules

1px cream-dark rule between each item. Creates rhythm and scannability without any decorative element. Used by Weta Workshop, FT, and quality editorial sites.

Option C

Bold Lead Word

First word or phrase bolded. Typographic hierarchy does the visual work — no symbols needed. DOC NZ fact sheets and NZ Govt design system use this.

Requires content edits

Option D

Row Rules + Single Amber Track

One amber left border on the whole list block (not per item), then hairline rules inside. The amber is structural — it frames the content, doesn't decorate each line.

Use sparingly

Recommendation B (row rules) for facility and feature lists — enough structure to aid scanning, zero decoration. A (pure spacing) works in card contexts where the card border already frames the list. C is the strongest typographically but requires reformatting list content — worth it for the lodge and range pages. D is too decorative for general use; save it for a single prominent call-out if needed.
